``` ## About The Los Gatos Think Tank is an independent, citizen-led research effort. We don't sell consulting and we don't take government contracts — the goal is simply to put the evidence behind local policy choices in front of the people those choices affect. Every project here uses public data sources (Census, GTFS, SWITRS, NTD, EPA), published methodologies (FTA cost-benefit guidelines, USDOT BCA Guidance, OMB Circular A-94), and code that anyone can inspect. If you find a mistake, tell us and we'll fix it in the open. Want to help, propose a topic, or provide input on our projects? Contact [elijah.asheghian@icloud.com](mailto:elijah.asheghian@icloud.com).
